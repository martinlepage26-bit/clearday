# Clearday — Perimenopause Support (Web Landing Page)

Static landing page for Clearday, an evidence-based perimenopause education and symptom-tracking app.

**Clearday is explicitly non-diagnostic and non-prescriptive.** It helps users track symptoms, access evidence-based education, and prepare for clinical consultations. It does not diagnose, prescribe, or replace medical care.

## Stack

- Static HTML/CSS/JS (Vite build output via Perplexity Pages)
- Hosted as a static site (Cloudflare Pages or equivalent)
- Embeddable via iframe from Perplexity Pages

## Structure

```
index.html          — Single-page entry point with Perplexity inline-edit bridge
favicon.svg         — SVG favicon (brand green #2D6A4F)
assets/
  index-*.js        — Bundled application JS (Vite build artifact)
  index-*.css       — Bundled styles (Vite build artifact)
.gitignore          — Excludes .wrangler/, node_modules/, dist/, .env
```

## Brand

- **Primary color:** `#2D6A4F`
- **Font:** Inter (300/400/500/600)
- **Bundle:** `app.clearday.mobile` (mobile app)

## Related Repositories

- **clearday-mobile**: Expo 56 / React Native 0.85 mobile app (symptom tracking, AI education assistant, cycle logging, health summaries)

## Development

This repo contains pre-built static assets generated by Perplexity Pages. The JS/CSS bundles are committed directly — there is no local build step.

To serve locally:

```bash
npx serve .
```

## Clinical Safety

This landing page describes an educational tool. All public-facing copy must:

- Never claim diagnostic or prescriptive capability
- Include the safety footer: "educational tool; does not diagnose, prescribe, or replace medical care"
- Describe symptom patterns as descriptive only, never diagnostic
